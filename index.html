Politique de Confidentialité – DomuTrack

Dernière mise à jour : 29/11/2025

1. Données collectées

DomuTrack ne collecte aucune donnée personnelle identifiable (nom, email, adresse, photos...).

Toutes les données de projets, budgets et factures sont stockées uniquement sur votre appareil.

2. Services tiers utilisés

✔ Google AdMob

Utilisé pour afficher des publicités. Peut collecter des données techniques anonymisées telles que l’identifiant publicitaire, les informations sur le téléphone ou l’usage des annonces.

Documentation AdMob

✔ Firebase Analytics

Utilisé pour analyser l’usage global de l’application (pages ouvertes, clics, durée d’utilisation). Les données sont anonymisées et ne permettent pas d’identifier l’utilisateur.

Documentation Firebase

✔ Firebase Crashlytics

Utilisé pour détecter et corriger les crashs. Les données incluent les informations techniques (stacktrace, modèle du téléphone, version Android).

✔ Firebase Performance Monitoring

Utilisé pour mesurer automatiquement les performances de l’application (temps de chargement des écrans, stabilité, fluidité). Les données sont techniques et anonymisées.

3. Stockage local des données

Toutes les données saisies dans DomuTrack (factures, lots, devis, budgets) sont stockées localement sur votre appareil et ne sont jamais envoyées sur un serveur externe.

4. Partage de données

DomuTrack ne vend ni ne partage aucune donnée personnelle. Seuls les services listés ci-dessus peuvent traiter des données techniques anonymisées afin de fonctionner.

5. Sécurité

Les données sont protégées via le stockage interne d’Android. Vous pouvez supprimer toutes les données en désinstallant l’application.

6. Vos droits

Puisque DomuTrack ne collecte aucune donnée personnelle, aucune demande de suppression ou de modification n’est nécessaire.

7. Modifications

Cette politique pourra être mise à jour pour refléter l’évolution de l’application ou la réglementation.
